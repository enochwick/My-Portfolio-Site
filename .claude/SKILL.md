# Deploy to Vercel

Run the production build, verify no errors, then deploy.

## Steps
1. Run `npm run build` and confirm zero errors
2. Run `npx vercel --prod` to deploy
3. Output the live URL

## Pre-deploy checklist
- All images have alt text
- No console errors in build output
- OG meta tags are set (title, description, image)
- Favicon is in place
